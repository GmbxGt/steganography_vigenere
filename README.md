НАЗНАЧЕНИЕ ПРОЕКТА
-----------------
Проект предназначен для демонстрации и практического использования
стеганографии в текстовых файлах совместно с шифром Виженера.

Программа позволяет:
1. Зашифровать текст шифром Виженера.
2. Преобразовать зашифрованный текст в двоичный вид.
3. Скрыть биты в обычном текстовом файле при помощи невидимых Unicode-
   символов (zero-width).
4. Извлечь скрытое сообщение и расшифровать его.

Визуально файл-контейнер выглядит почти так же, как обычный текстовый файл.

CLI — КОМАНДНАЯ СТРОКА
---------------------

Скрытие сообщения:
    python main.py hide -c container.txt -s secret.txt -k KEY

Результат:
    stego_container.txt

Извлечение сообщения:
    python main.py extract -i stego_container.txt -k KEY

Результат:
    extracted_secret.txt

====================================================================

АРХИТЕКТУРА ПРОЕКТА
------------------

steganography.py
  - скрытие и извлечение битов с помощью zero-width символов
  - упаковка текста с заголовком MAGIC + длина

vigenere.py
  - шифрование и дешифрование шифром Виженера
  - расширенный алфавит (латиница + кириллица)

utils.py
  - чтение и запись файлов (UTF-8)
  - проверка ключа

main.py
  - CLI интерфейс

gui.py
  - GUI интерфейс (Tkinter)

run.py
  - меню выбора режима

====================================================================

ПРИМЕР ИСПОЛЬЗОВАНИЯ МОДУЛЕЙ В КОДЕ
---------------------------------

from steganography import text_to_binary, binary_to_text
from vigenere import vigenere_encrypt, vigenere_decrypt

encrypted = vigenere_encrypt("Секрет", "КЛЮЧ")
bits = text_to_binary(encrypted)
restored = binary_to_text(bits)
decrypted = vigenere_decrypt(restored, "КЛЮЧ")
