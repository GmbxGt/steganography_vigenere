steganography

module documentation

Стеганография на основе невидимых символов.

Модуль реализует скрытие бинарных данных в «контейнер» (строку) путём вставки нулевой ширины символов (Zero Width Space / Zero Width Non-Joiner) после пробелов. Также поддерживаются преобразования текста в бинарный вид и обратно.

def hide_text_in_container(secret: str, container: str) -> str: ¶

Скрывает бинарные данные в контейнере.

Для каждого пробела контейнера, начиная с начала строки, может быть использован один бит. Бит 0 кодируется символом ZWS_0, бит 1 — символом ZWNJ_1, которые вставляются сразу после пробела.

Parameters
secret:strБинарная строка ('0'/'1'), которую требуется скрыть.
container:strТекст-контейнер, содержащий достаточное число пробелов.
Returns
strНовая строка-контейнер со скрытыми данными.
Raises
ValueErrorЕсли контейнер не содержит достаточного количества пробелов для размещения всех битов.
