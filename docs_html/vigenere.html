Шифр Виженера.

Модуль реализует шифрование и расшифрование текста по алгоритму Виженера для расширенного алфавита проекта (латиница + кириллица и некоторые спецсимволы).

Шифрует текст шифром Виженера.

Текст приводится к верхнему регистру. Символы, отсутствующие в алфавите, не шифруются и переносятся в результат без изменений.

def _prepare_key(key: str): ¶

Подготавливает ключ для шифра Виженера.

Функция приводит ключ к верхнему регистру и удаляет символы, которых нет в алфавите. Возвращает список символов ключа и индексную таблицу алфавита.

Parameters
key:strИсходный ключ.
Returns
tuple[list[str], dict[str, int], int]Кортеж (key_filtered, index, n), где key_filtered — список допустимых символов ключа, index — отображение символ -> позиция в алфавите, n — длина алфавита.
Raises
ValueErrorЕсли ключ пустой.
ValueErrorЕсли после фильтрации ключ не содержит допустимых символов.
