class SteganographyApp:

View In Hierarchy

Главное окно приложения стеганографии.

Экран позволяет: - зашифровать сообщение (шифр Виженера) и скрыть его в текстовом контейнере; - извлечь скрытое сообщение и расшифровать его; - управлять файлами контейнера/результата.
